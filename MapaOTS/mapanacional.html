Mapa Nacional con Datamaps

Estructura HTML

DOCTYPE y HTML: Se declara el tipo de documento y se establece el idioma de la página.
Meta Tags: Se define la codificación de caracteres y se ajusta la vista para dispositivos móviles.
Title: Se establece el título que aparecerá en la pestaña del navegador.
Bootstrap: Se incluye Bootstrap para el diseño básico y la estructura responsiva.
Container: Se crea un contenedor donde se dibuja el mapa, con dimensiones específicas y un borde.

Etiquetas Script

D3.js: Se carga la biblioteca D3.js, que es fundamental para la manipulación de datos y gráficos SVG.
TopoJSON: Se carga TopoJSON, que es un formato para representar datos geográficos y se utiliza para crear mapas.
DataMaps:Se carga la biblioteca DataMaps, que facilita la creación de mapas interactivos.

Contenido de mapanacional.js

Serie de Datos: Se define un array series que contiene pares de valores. Cada par representa un departamento de Colombia con su respectivo código ISO y un número que podría representar, por ejemplo, la cantidad de colaboradores.

Creación del Dataset

Dataset: Se inicializa un objeto vacío dataset donde se almacenarán los datos formateados para el mapa.
Cálculo de Valores Mínimos y Máximos: Se extraen los valores de la serie y se calcula el mínimo y máximo para definir el rango de colores.
Escala de Color: Se define una escala de color lineal con D3.js que asigna colores desde el más claro hasta el más oscuro basado en los valores.
Llenado del Dataset: Se itera sobre cada elemento de series y se llena dataset con la información estructurada que incluye el número de cosas y el color de relleno

Configuración y Creación del Mapa

Instanciación de DataMap: Se crea una nueva instancia de Datamap, especificando el elemento HTML donde se renderizará el mapa y el ámbito geográfico (Colombia).
Proyección: Se establece una proyección geográfica utilizando Mercator, centrando el mapa en las coordenadas específicas de Colombia y ajustando la escala.
Datos: Se pasa el dataset previamente creado, que contiene la información para cada departamento. Configuración de Geografía: Se configuran opciones como el color del borde, el comportamiento al resaltar regiones y la plantilla del popup que se muestra al interactuar con el mapa. El popup muestra el nombre de la región y el número de colaboradores

Comentarios y Opciones Adicionales

Comentarios: Se incluye un comentario sobre la opción de mostrar etiquetas para los nombres de los departamentos. Esta línea está comentada, pero podría descomentarse para mostrar etiquetas en el mapa.

DataMaps

DataMaps es una biblioteca de visualización de datos que permite la creación de mapas interactivos utilizando SVG (Scalable Vector Graphics) en la web. Se basa en D3.js, una de las bibliotecas más populares para la manipulación de documentos basados en datos. Esto le otorga a DataMaps una gran flexibilidad y potencia, facilitando la visualización de datos geoespaciales de manera clara y atractiva.
